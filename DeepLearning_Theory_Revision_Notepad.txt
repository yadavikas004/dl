UNIT I — Applied Math & ML Basics (Linear Algebra + Numerical Computation)
============================================================

KEY POINTERS TO REVISE
- Scalars / vectors / matrices / tensors: shapes, indexing, broadcasting idea.
- Matrix–vector and matrix–matrix multiplication: dimension rules; associative; NOT commutative.
- Identity, inverse: when inverse exists (square + full rank); A A^{-1} = I.
- Linear dependence/independence: span; basis; rank.
- Norms: L0 (count), L1, L2, Linf; Frobenius norm for matrices.
- Special matrices/vectors: diagonal, symmetric, orthogonal, PSD; one-hot vectors.
- Eigen decomposition (esp. symmetric): eigenvalues/vectors; interpretation.
- Overflow/underflow: floating point limits; log tricks for stability.
- Poor conditioning: condition number; sensitivity of solutions/gradients.
- Gradient-based optimization: GD/SGD concept; learning rate; convergence intuition.
- Constraint optimization: Lagrange multipliers; KKT idea (high level).

DEFINITION QUESTIONS (CRISP ANSWERS)
1) Define tensor.
   Multidimensional array generalizing scalars (0D), vectors (1D), matrices (2D).

2) Define span.
   Set of all linear combinations of given vectors.

3) Define linear dependence.
   Vectors are dependent if some nontrivial coefficients produce a linear combination equal to zero.

4) Define rank.
   Dimension of column space (number of independent columns); equals number of nonzero singular values.

5) Define orthogonal matrix.
   Q^T Q = I; columns are orthonormal; Q^{-1} = Q^T.

6) Define PSD matrix.
   Symmetric A with x^T A x >= 0 for all x.

7) Define eigenvalue/eigenvector.
   Av = λv for nonzero v.

8) Define condition number.
   Sensitivity measure; for invertible A: κ(A) = ||A|| · ||A^{-1}||.

9) Define overflow/underflow.
   Overflow: numbers exceed max representable. Underflow: very small magnitudes round to 0/subnormal.

MATH / FORMULAS (HIGH-YIELD)
- Multiplication shapes: (m×n)(n×p) = (m×p)
- Inverse existence: A^{-1} exists iff A is square and det(A) != 0 (equivalently full rank).
- Norms:
    ||x||_1 = Σ_i |x_i|
    ||x||_2 = sqrt(Σ_i x_i^2)
    ||x||_∞ = max_i |x_i|
    ||A||_F = sqrt(Σ_{i,j} a_{ij}^2)
- Eigen facts (symmetric A): real eigenvalues; orthonormal eigenvectors; A = Q Λ Q^T.
- Log-sum-exp trick:
    log Σ_i exp(z_i) = m + log Σ_i exp(z_i - m), where m = max_i z_i
- Gradient Descent update:
    θ_{t+1} = θ_t - η ∇_θ L(θ_t)
- Lagrangian for equality constraint g(x)=0:
    L(x,λ) = f(x) + λ g(x)

UNIT II — Deep Networks, Feedforward, Regularization, Optimization
============================================================

KEY POINTERS TO REVISE
- Neuron + activation: nonlinearity is required.
- Feedforward network: layers, parameters, forward pass, loss.
- Backprop idea: chain rule through computational graph.
- Vanishing/exploding gradients: deeper nets; activation choice; initialization.
- Regularization: L2 (weight decay), L1 (sparsity), dropout, early stopping, data augmentation (concept).
- Optimization: SGD, momentum, RMSProp/Adam (mechanics + intuition); LR schedules.
- Batch norm: what it normalizes; training vs inference difference.

DEFINITION QUESTIONS (CRISP ANSWERS)
1) Define deep feedforward network.
   Composition of affine transforms + nonlinearities mapping input to output without cycles.

2) Define backpropagation.
   Efficient gradient computation using chain rule from output back to parameters.

3) Define overfitting.
   Good training performance but poor generalization due to fitting noise/over-complexity.

4) Define weight decay.
   L2 penalty on weights; tends to shrink weights during training.

5) Define dropout.
   Randomly zeroing activations during training to reduce co-adaptation and overfitting.

6) Define momentum.
   Exponentially averaged gradient direction to speed up and smooth SGD.

MATH / FORMULAS (HIGH-YIELD)
- Forward layer:
    a^(l) = φ(W^(l) a^(l-1) + b^(l))
- Activations:
    ReLU(x) = max(0,x)
    sigmoid(x) = 1 / (1 + e^{-x})
    tanh(x) = (e^x - e^{-x}) / (e^x + e^{-x})
- Losses:
    MSE = (1/N) Σ (y - y_hat)^2
    Softmax: p_k = exp(z_k) / Σ_j exp(z_j)
    Cross-entropy: L = - Σ_k y_k log p_k
- L2-regularized objective:
    L'(W) = L(W) + (λ/2) ||W||_2^2
- SGD with momentum:
    v ← β v + ∇L
    θ ← θ - η v

UNIT III — Convolutional Networks, Sequence Modeling, Applications
============================================================

KEY POINTERS TO REVISE
- Convolution: kernels/filters, stride, padding, channels.
- Feature maps: local receptive fields; weight sharing.
- Pooling: max/avg; invariance vs information loss.
- Typical CNN blocks: conv → (bn) → relu → pool → ... → flatten → FC.
- Sequence modeling: RNN idea; BPTT; vanishing gradients; LSTM/GRU motivation.
- Applications: image classification, detection (high level), NLP tasks (classification/translation).

DEFINITION QUESTIONS (CRISP ANSWERS)
1) Define convolution (in deep learning).
   Sliding dot product of kernel weights with local input patches producing a feature map.

2) Define receptive field.
   Input region influencing a particular activation.

3) Define padding.
   Adding borders to control spatial size and preserve edge information.

4) Define stride.
   Step size of kernel movement across input.

5) Define RNN.
   Network with recurrent state h_t capturing past information in sequences.

6) Why LSTM/GRU?
   To mitigate vanishing gradients and capture long-term dependencies.

MATH / FORMULAS (HIGH-YIELD)
- Conv output size (per spatial dimension):
    out = floor((n + 2p - k)/s) + 1
  where n=input size, p=padding, k=kernel size, s=stride

- RNN equations:
    h_t = φ(W_x x_t + W_h h_{t-1} + b)
    y_hat_t = g(W_y h_t + c)

- BPTT intuition:
    Gradients multiply through time-step Jacobians → can vanish/explode.

- Generative models: VAE basics; GAN basics; likelihood vs implicit generation.

DEFINITION QUESTIONS (CRISP ANSWERS)
1) Define autoencoder.
   Network trained to reconstruct input via compressed latent vector z.

2) Define latent representation.
   Lower-dimensional features capturing key structure of data.

3) Define variational inference.
   Approximate an intractable posterior with a tractable family by optimizing a bound.

4) Define ELBO (VAE).
   Lower bound on log-likelihood used as training objective.

5) Define GAN.
   Generator and discriminator trained adversarially so generator outputs look like real data.

MATH / FORMULAS (HIGH-YIELD)
- PCA objective (idea):
    Maximize variance / minimize reconstruction error under orthonormal constraints.

- Autoencoder loss (typical):
    L = (1/N) Σ_i ||x_i - x_hat_i||^2   (or BCE for binary data)

- VAE ELBO:
    log p(x) >= E_{q(z|x)}[log p(x|z)] - KL(q(z|x) || p(z))

- GAN minimax:
    min_G max_D  E_{x~pdata}[log D(x)] + E_{z~p(z)}[log(1 - D(G(z)))]

DEFINITION QUESTIONS (KEEP ANSWERS SHORT)
- What is a scalar / vector / matrix / tensor?
  Single number / ordered 1-D array / 2-D rectangular array / higher-order array of numbers.
- What is matrix transpose?
  Operation that flips a matrix over its diagonal; element (i,j) becomes (j,i).
- What is identity matrix?
  Square matrix with ones on main diagonal and zeros elsewhere, such that IA = AI = A.
- Define span of a set of vectors.
  Set of all linear combinations of those vectors; equals column space for matrix columns.
- Define linearly independent vectors.
  Vectors for which only the trivial combination a_i = 0 gives Σ a_i v_i = 0.
- Define a norm and list three properties.
  Function measuring vector length that satisfies positive definiteness, homogeneity, and triangle inequality.
- Define condition number of a matrix.
  Ratio of largest to smallest singular value/eigenvalue; measures sensitivity of solutions.
- Define numerical overflow and underflow.
  Overflow: result too large to represent; underflow: result too close to zero to represent accurately.
- Define gradient descent.
  Iterative method that updates parameters in negative gradient direction to minimize a differentiable function.

DEFINITION QUESTIONS
- Define Deep Neural Network (DNN).
  Artificial neural network with multiple hidden layers between input and output performing hierarchical representation learning.
- Define softmax function.
  Function converting real-valued scores into a probability distribution whose components are positive and sum to 1.
- What is cross-entropy loss?
  Loss measuring difference between true and predicted probability distributions; used widely for classification.
- What is overfitting in deep learning?
  Model fits training data (including noise) very well but performs poorly on unseen data.
- Define L2 and L1 regularization.
  L2: penalty on sum of squared weights; L1: penalty on sum of absolute weights producing sparse solutions.
- What is dropout?
  Regularization method that randomly disables neurons during training with fixed probability.
- Define batch, stochastic, and mini-batch gradient descent.
  Batch: uses full dataset per update; stochastic: one example; mini-batch: small subset per update.
- What is momentum in optimization?
  Technique that accumulates an exponential moving average of past gradients to accelerate updates in consistent directions.
- Define empirical risk minimization (ERM).
  Principle of minimizing average loss over training set as an approximation of minimizing expected loss over true distribution.

DEFINITION QUESTIONS
- Define Convolutional Neural Network (CNN).
  Neural network architecture using convolution and pooling layers to process grid-like data such as images.
- What is a feature map?
  Output of a convolutional layer obtained by sliding a filter over the input and computing dot products.
- Define stride and padding.
  Stride: number of pixels the filter moves each step; padding: extra pixels (often zeros) added around input border.
- What is pooling in CNN?
  Downsampling operation (e.g., max/average) that summarizes features in local neighborhoods to reduce size and overfitting.
- Define Recurrent Neural Network (RNN).
  Neural network with cyclic connections that maintain a hidden state to capture sequence dependence.
- What is sequence classification?
  Task where a whole input sequence is mapped to a single label, such as sentiment of a sentence.
- Define word embedding.
  Dense low-dimensional vector representation of words capturing semantic similarity.
- What is data augmentation in computer vision?
  Generating extra training images by transformations such as flip, rotation, crop, zoom to improve generalization.

------------------------------------------------------------

POINTERS
- Unsupervised learning motivation: structure discovery without labels; high-dimensional data challenges.
- Linear factor models: PPCA, Factor Analysis, ICA, Sparse Coding, SFA; latent variables and manifolds.
- Autoencoders: encoder–code–decoder structure; undercomplete vs regularized vs denoising; main applications.
- Representation learning: idea of learning good internal features that simplify downstream tasks.
- Approximate inference: need in Bayesian learning; posterior intractability; stochastic (MCMC) vs deterministic (variational) methods.

- Deep generative models: generative vs discriminative; supervised vs unsupervised; DGMs for sample generation.
- GANs: generator–discriminator game, equilibrium idea, conditional GANs, CNN-based GANs.
- Boltzmann Machines, Restricted Boltzmann Machines, Deep Belief Networks; energy-based view and layer-wise pretraining.

DEFINITION QUESTIONS
- Define linear factor model.
  Probabilistic model where observed variables are expressed as linear combinations of latent factors plus noise.
- Define Probabilistic PCA (PPCA).
  PCA variant that assumes data generated from Gaussian latent variables with isotropic Gaussian noise, giving PCA a probabilistic view.
- What is an autoencoder?
  Neural network trained to reconstruct its input via a compressed latent representation.
- Define representation learning.
  Process of automatically learning useful feature representations from data instead of using hand-crafted features.
- Define posterior probability.
  Updated probability of parameters or hypotheses after observing data, obtained via Bayes’ theorem.
- What is approximate inference?
  Family of methods that approximate an intractable posterior using sampling or optimization (e.g., MCMC, variational inference).
- Define ELBO.
  Evidence Lower Bound, an objective that lower-bounds log-likelihood and is maximized in variational inference.
- Define Generative Adversarial Network (GAN).
  Generative model with a generator and discriminator trained adversarially so generated samples resemble real data.
- What is a Restricted Boltzmann Machine (RBM)?
  Stochastic energy-based model with visible and hidden units and no intra-layer connections, trained to reconstruct data.
